# 🎨 Design Practice

A personal sandbox repository for practicing and experimenting with front-end web design concepts.

## 📌 About

This repo is a collection of design exercises and mini-projects built while learning and practicing web design fundamentals and frameworks.

## 🛠️ Tech Stack

- **HTML5** – Structure and markup
- **CSS3** – Styling and layouts
- **SCSS** – Modular and maintainable styling
- **Tailwind CSS** – Utility-first CSS framework
- **Bootstrap** – Pre-built responsive components
- **JavaScript** – Interactivity and DOM manipulation

## 🎯 Goals

- Strengthen core HTML/CSS fundamentals
- Get comfortable with SCSS architecture
- Build responsive UIs using Tailwind CSS
- Practice component-based design with Bootstrap
- Add interactivity using vanilla JavaScript

## 🚀 How to Use

1. Clone the repo
```bash
   git clone https://github.com/<your-username>/design-practice.git
```
2. Navigate into a folder
```bash
   cd design-practice/tailwind
```
3. Open `index.html` in your browser or use Live Server
